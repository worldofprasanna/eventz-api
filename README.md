## Install

1. Install ruby, rails
2. To install dependencies,
```
bundle install
```
3. To setup the database,
```
bundle exec rails db:setup
```
4. To run the database migrations,
```
bundle exec rails db:migrate
```

## Usage

1. Start the server
```
bundle exec rails s
```
